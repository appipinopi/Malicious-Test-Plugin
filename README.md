# Malicious Test Plugin

このプラグインは、EDBPのプラグインシステムのテスト用に作成されたジョークプラグインです。

## 機能
1. **音声ループ**: 有効化すると「You are an idiot!!」の音声が無限ループ再生されます。
2. **アイコン増殖**: EDBPのアイコンが画面全体を埋め尽くすように出現し続けます。
3. **特大メッセージ**: 画面中央に警告文が表示されます。

## 注意事項
- **音量に注意**: 有効化する前にPCの音量を確認してください。
- **ブラウザの制限**: ページを読み込んだ直後に一度も画面をクリックしていない場合、ブラウザのセキュリティ設定により音声が流れないことがあります。その場合は画面のどこかをクリックしてください。
